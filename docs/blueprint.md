# **App Name**: Como Seu Centro Decisor Opera

## Core Features:

- Quiz Initialization & Flow Control: Manages the display of the welcome screen, guides users through questions, and transitions to the loading and results screens.
- Dynamic Question Presentation: Displays each of the 9 quiz questions and their respective answer choices one at a time, ensuring a smooth, single-page experience.
- Scoring Logic & Tie-Breaker: Calculates points for the first 5 decisive questions, assigning +1 point to the corresponding decision profile for each selected answer. Implements a tie-breaker rule using question 5 results if scores are tied.
- Quiz Progress Indicator: Visually indicates the user's progress through the quiz, such as 'Pergunta X de 9' or a simple progress bar.
- Loading Transition Screen: Presents a transition screen with the message 'Analisando seu padrão decisório...' and a simulated progress bar for approximately 2 seconds before revealing the final result.
- Profile Result Display: Upon completion, determines and displays the predominant decision-making profile (Explorador, Protetor, Visionário, or Estrategista) along with a concise descriptive text for that profile.

## Style Guidelines:

- Cor principal: Violeta espiritual (#6B4EFF) para evocar espiritualidade, consciência e reflexão interior.
- Cor secundária: Lavanda suave (#B8A6FF) para complementar a cor principal, contribuindo para uma paleta calma e contemplativa.
- Cor de fundo principal: Branco levemente frio (#F6F5FF) como base, criando uma tela limpa e etérea.
- Cor de destaque: Roxo profundo (#3D2C8D) para elementos importantes que necessitam de mais presença, como botões de ação ou títulos secundários.
- Cor neutra para textos: Cinza profundo (#2B2B2B), garantindo legibilidade e uma sensação moderna e sóbria.
- Cor suave para elementos secundários: Lavanda clara (#E9E6FF) para detalhes sutis e fundos de seções menos proeminentes.
- Uso de degradês suaves no fundo, como linear-gradient(135deg, #6B4EFF, #B8A6FF) ou transições entre #F6F5FF e #E9E6FF, para criar uma sensação de profundidade leve e etérea.
- Botões devem usar a cor violeta principal (#6B4EFF) com texto branco para clareza e elegância.
- Corpo e títulos com tipografia elegante e leve, que transmita a sensação de reflexão pessoal e espiritualidade, como 'Alegreya' ou similar.
- Ícones minimalistas e baseados em linhas, abstratos e sutis, que sugiram conceitos de consciência, caminhos ou horizontes, evitando qualquer aparência infantil ou 'gamificada'.
- Layout limpo, minimalista e espaçoso, enfatizando blocos de conteúdo únicos (tela de boas-vindas, pergunta, resultado). Elementos visuais com bordas arredondadas e sombras leves para suavidade. 
- Animações discretas e transições suaves, como fades lentos entre telas e perguntas, e pulsações gentis para indicadores de progresso, reforçando a calma e a progressão meditativa.
